# Prime Analyzer Qt

Aplicación de escritorio desarrollada con Python para identificar números primos y realizar su descomposición en factores primos.

> Este repositorio documenta la evolución de un proyecto académico hacia una aplicación de escritorio moderna utilizando Qt (PySide6), siguiendo buenas prácticas de desarrollo, Git y control de versiones.
